example.ts 
import JSONDatabase from 'jsondb-high';

// Initialize database
const db = new JSONDatabase('data.json');

// Write data
await db.set('user:1', {
  name: 'Alice',
  role: 'admin'
});

// Query with chainable API
const admins = await db.query('users')
  .where('role').eq('admin')
  .limit(10).exec();
Features

Built for Performance

Every feature engineered for speed, reliability, and developer experience.

Blazing Fast

Rust-powered core via N-API delivers native performance with 2M+ ops/sec reads.

~2M reads/sec ~260K writes/sec

Crash Recovery

Group Commit WAL with configurable durability modes for crash-safe writes.

Multi-Core

Adaptive parallelism using Rayon automatically utilizes all CPU cores.

O(1) Indexing

In-memory Map indices enable instant lookups by any field.

AES-256 Encryption

Optional AES-256-GCM encryption keeps your data secure at rest.

TTL Support

Auto-expire keys after a specified time, just like Redis.

Pub/Sub Events

EventEmitter-style subscriptions to monitor key changes in real-time.

Aggregations

Built-in sum, avg, min, max, groupBy, and distinct operations.

Zero Dependencies

Self-contained native binary. No external databases required.

Architecture

Hybrid Performance Model

Choose your consistency mode based on your durability needs.

In-Memory Mode

Maximum throughput at ~260k ops/sec. Best for caching and temporary data.

durability: 'none'

WAL Mode (Durable)

Crash-safe writes at ~80k ops/sec. Full persistence with recovery.

durability: 'batched'
