üéØ Key Insights

üî• Read Operations

get and has are extremely fast (~1.5M-2M ops/s) since they only access in-memory data. WAL mode is now equally fast!

‚úçÔ∏è Write Operations

set and delete achieve ~260k+ ops/s. With Group Commit WAL, durable mode is almost as fast as in-memory.

üîç Index Lookups

findByIndex provides O(1) performance (500k+ ops/s) regardless of dataset size - perfect for large collections.

üì¶ Batch Operations

Batching multiple writes into single calls provides 187k ops/s - ideal for bulk imports.
